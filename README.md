# Predicting Star-Level Contracts in the NBA

This project's goal is to determine which early career performance factors lead to a "star-level" contract during a player's career.

Definitions:
- A "star-level" contract is one that is greater than 20% of the salary cap for the given year
- MP_pg = minutes played per game on average
- PTS = points
- TRB = total rebounds / TRB% = percentage of available rebounds player grabbed
- AST = assists / AST% = percentage of team field goals player assisted in
- STL = steals / STL% = percentage of opponent possessions that ended in a steal by player
